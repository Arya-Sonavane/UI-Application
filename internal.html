Access Specifiers

private : private access specifier means we can cannot access class member outside of class. public: public is access specifier which is used for access class member outside of class or outside of package.
protected: protected class member is only accessible in child class default : default class member can accessible outside of class but within same package. class is keyword for class declaration FirstDec is classname and user can give any name to his class. public static void main(String x[]): it is main function of java same like as main function in C or CP.
